# Pretendard 폰트

전역 폰트로 사용. `pubspec.yaml`의 `fonts:` 블록에 등록되어 있고 `AppTheme`에서 `fontFamily: 'Pretendard'`로 적용됨.

## 포함 파일 (Pretendard 1.3.9, static/alternative)
- `Pretendard-Regular.ttf` (400)
- `Pretendard-Medium.ttf` (500)
- `Pretendard-SemiBold.ttf` (600)
- `Pretendard-Bold.ttf` (700)
